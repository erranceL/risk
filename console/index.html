Risk Console

Nicevent 赔率风控台

基于 1s 自建指数价回测：0.2 币安现货 + 0.8 币安永续。你手动设置的 edge 才是生效值，系统给出的建议 edge 只作参考。

你可以控制什么？

先选产品和周期，再把人工 edge 应用到当前筛选；目标风险和最低平台 return 会影响“建议 edge”。

怎么看这些指标？

人工 edge你手动设定的平台抽水/安全垫，是当前页面真正采用的 edge。
建议 edge系统在候选 edge 里找“平台 return 达标且风险等级不超过目标”的最小值，只是建议。
平台 return回测里平台赚/亏占成交本金的比例；小于 0 代表这组赔率下平台亏。
风险等级1-10 分，越高越危险；综合赔率压力、方向失衡、最大亏损和波动。
用户胜率随机方向订单在这段历史价格上的真实胜率，和 edge 本身无关。
护栏状态赔率被 floor/ceiling/publish 上下限卡住时会提示，说明 edge 不一定完全转化成赔率。
